Aula 2 - O que é Lógica

Lógica = É a forma correta organizada de pensar para chegar a uma conclusão correta. 

- organizar idéias
- tomar decisão (organizada)
- Criar soluções passo a passo (Resolver problemas passo a passo)

Sequencia lógica: É a ordem correta dos passos que precisam acontecer para que algo faça sentido ou funcione: 

OU 

é a organização de ações na ordem certa para chegar a um resultado. 

LÓGICA DE PROGRAMAÇÃO

É organizar as instruções na ordem certa para que o computador execute a tarefa;

*O COMPUTADOR NÃO PENSA/ NÃO INTERPRETA EMOÇÕES*


____________________________________________________________________________________________________________________________
três passos da base de Programação

Sequencia - Ordem dos passos
Decisão - Se isso acontecer, faça aquilo
Repetição - Faça várias vezes. 

COMPUTAR EXECUTA TAREFA POR TAREFA

Algoritmo > Conjunto de passos para resolver um problema (Traduzindo, uma receita de bolo, que é preciso seguir passo a passo, medidas e tals)

Variáveis > Espaço para guardar informações (Nomeando como vai chamar aquele pedaço da memoria) 

Condição - é a decisão (Comparação)

Laços de Repetição: Indica quantas vezes deve repetir

COnstante - São informações que não mudam

Todo Algoritmo precisa de: 
Inicio
Dados de entrada
Processamento
Decisão
Saída
Fim

EXERCICIO 

Uma loja online oferece desconto para seus clientes com base nas seguintes regras:

O cliente recebe 10% de desconto se o valor total da compra for maior que R$ 200.

O cliente recebe 15% de desconto se for cliente VIP.

Caso o cliente seja VIP e o valor da compra seja maior que R$ 200, o desconto aplicado será de 20%.

Caso nenhuma das condições anteriores seja atendida, não haverá desconto.

Seu desafio

Identifique as variáveis necessárias para resolver o problema.

Defina as condições lógicas envolvidas.

Monte a estrutura lógica (em português estruturado ou fluxograma) que represente essa regra de negócio.

Explique qual será o desconto aplicado nos seguintes casos:

Cliente VIP com compra de R$ 250

Cliente não VIP com compra de R$ 250

Cliente VIP com compra de R$ 150

Cliente não VIP com compra de R$ 150
